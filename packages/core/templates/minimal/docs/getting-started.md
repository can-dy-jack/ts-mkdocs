# Getting Started

## Installation

```bash
npm install ts-mkdocs
```

## Create a new project

```bash
npx ts-mkdocs new my-docs
cd my-docs
```

## Start the dev server

```bash
npx ts-mkdocs serve
```

Open [http://127.0.0.1:8000](http://127.0.0.1:8000) in your browser.

## Build for production

```bash
npx ts-mkdocs build
```

The static site will be output to the `site/` directory.
