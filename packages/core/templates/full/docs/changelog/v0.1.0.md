# Version 0.1.0

**Release Date:** 2024-06-25

Initial release of ts-mkdocs ‚Äî a TypeScript implementation of MkDocs with Material theme.

## Core Features

### CLI Commands

- `ts-mkdocs new` ‚Äî Project scaffolding with starter config and docs
- `ts-mkdocs build` ‚Äî Static site generation from Markdown files
- `ts-mkdocs serve` ‚Äî Development server with live reload (SSE-based)

### Material Theme

- Responsive two-column layout
- Dark/light mode toggle with system preference detection
- Sticky header with navigation
- Customizable color palette
- Font options (system fonts or Google Fonts)
- Reader settings panel (colors, fonts, font size)

### Navigation

- Explicit `nav:` configuration in `ts-mkdocs.yml`
- Auto-inferred navigation from file tree
- Top-level sections as tabs
- Sidebar navigation with sections
- Footer navigation (previous/next page links)
- Back-to-top floating button

### Content Features

- Table of contents extracted from headings
- Per-page TOC with sticky sidebar
- Syntax highlighting via Shiki
- Dual light/dark syntax themes
- Copy button on code blocks
- Line numbers on code blocks
- Language labels on code blocks
- Code wrapping support

## Markdown Extensions

### Basic Extensions

- **Admonitions** ‚Äî `!!!` / `???` callout blocks with icons and collapse
- **Content Tabs** ‚Äî `===` syntax for tabbed content
- **Enhanced Code Blocks** ‚Äî Superfences with syntax highlighting
- **Task Lists** ‚Äî GitHub-style checkboxes `- [ ]` / `- [x]`
- **Footnotes** ‚Äî Reference notes with `[^1]` syntax
- **Abbreviations** ‚Äî Tooltips for terms with `*[ABBR]:` definitions
- **Definition Lists** ‚Äî Term/definition pairs
- **Icons** ‚Äî Shortcodes like `:material-home:` for 10,000+ icons
- **Keyboard Keys** ‚Äî `++Ctrl+C++` syntax for key combinations
- **Highlight** ‚Äî `==marked text==` for emphasis
- **Emoji** ‚Äî `:smile:` shortcodes with full/light index
- **MagicLink** ‚Äî Auto-link issues, PRs, commits, and URLs
- **Attributes** ‚Äî `{: .class #id}` syntax for element attributes

### Advanced Extensions

- **Math** ‚Äî LaTeX equations via KaTeX or MathJax
- **Mermaid** ‚Äî Diagrams from code fences (flowcharts, sequences, Gantt)
- **Superscript/Subscript** ‚Äî `^sup^` and `~sub~` syntax
- **Insert/Delete** ‚Äî `^^inserted^^` and `~~deleted~~` for tracked changes

## Plugins

### Built-in Plugins

- `search` ‚Äî Client-side search powered by Lunr.js
- `tags` ‚Äî Tag system for categorizing pages
- `social` ‚Äî Social card meta tags (Open Graph, Twitter Cards)
- `meta` ‚Äî Meta tag support from `.meta.yml` files
- `sitemap` ‚Äî Sitemap.xml generation
- `robots` ‚Äî Robots.txt generation
- `git-revision-date` ‚Äî Git-based creation and update dates

### Plugin System

- Event-driven plugin architecture
- Lifecycle hooks: `on_config`, `on_files`, `on_nav`, `on_page_markdown`, `on_page_content`, `on_post_build`
- External plugin support via npm packages

## SEO & Metadata

- Open Graph meta tags
- Twitter Card support
- Canonical URLs
- Custom HTML meta tags
- Article license declarations (CC BY-NC-SA 4.0)

## Developer Features

### Frontmatter

- Per-page metadata: `title`, `description`, `tags`, `authors`, `dates`
- Search boost and exclude options
- Custom edit links
- Reading time calculation
- Author profiles with avatars and social links

### Comments

- Giscus integration (GitHub Discussions)
- Utterances integration (GitHub Issues)
- Per-page comment enable/disable

### Other Features

- Custom CSS and JavaScript support
- 404 page
- Mobile responsive design
- Instant navigation (SPA-like)
- Prefetch on hover
- Progress bar on navigation
- Image lightbox zoom
- Share buttons (X, Facebook, Weibo, LinkedIn, etc.)

## Technical Details

- **Language:** TypeScript (ESM-only, Node 20+)
- **Package Manager:** pnpm 9
- **Build Tool:** tsup
- **Test Framework:** Vitest
- **Template Engine:** Nunjucks
- **Markdown Parser:** markdown-it with extensions
